    <h1>Color Flipper</h1>
    <p>Esse é um projeto simples que muda a cor de fundo de uma página com um clique. A cor é gerada de forma aleatória a partir de hexadecimais.</p>
    <h2>Instalação</h2>
    <p>Baixe os arquivos HTML, CSS e JavaScript e abra o arquivo HTML em um navegador.</p>
    <h2>Como usar</h2>
    <p>Clique no botão "Clique aqui" e a cor de fundo da página será alterada para uma cor gerada aleatoriamente. A cor gerada é exibida como texto na tela.</p>
